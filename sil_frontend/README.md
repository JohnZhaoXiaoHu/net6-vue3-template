# template_frontend

Frontend Vue project, connected to template .NET Core 6 backend
## Project setup
```
npm install
```
Create .env.local file populated with the following variables
```
VUE_APP_BASE_URL=https://localhost:5001/api
```
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
